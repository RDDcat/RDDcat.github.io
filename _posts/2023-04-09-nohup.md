## jar 백그라운드 실행
### 📖 기억


sudo nohup java -jar 파일이름.jar &

예시

>sudo nohup java -jar discovery.jar &
>
>sudo nohup java -jar apigateway.jar &
>
>sudo nohup java -jar backend.jar &

포트 상태 확인

>sudo netstat -tnlp
